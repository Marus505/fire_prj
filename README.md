## 📊 **나만의 SOXL 단타 매매법**

### 🎯 **전략 개요**
- **대상 종목**: SOXL (반도체 3배 레버리지 ETF)
- **계좌 구성**: 20개 계좌로 분할
- **자본 배분**: 동일한 시드를 20분할
- **매매 방식**: 등차수열 기반 LOC(지정가) 매수/매도

---

### 💰 **자본 관리**

#### **계좌 구성**
```
총 자본: $20,000
계좌 수: 20개
계좌당 자본: $1,000
```

#### **자본 분할 원칙**
- 각 계좌에 동일한 시드 배분
- 리스크 분산을 통한 안정성 확보
- 개별 계좌 손실 시 전체 영향 최소화

---

### 📈 **매수 전략**

#### **1일차 매수**
1. **1번 계좌**: 현재가 전액 매수
2. **2~20번 계좌**: 1번 매수단가 기준 등차수열 LOC 매수

#### **등차수열 매수 가격**
```
1번 계좌: $20.00 (현재가 매수)
2번 계좌: $19.70 (1번 - 0.3)
3번 계좌: $19.40 (1번 - 0.6)
4번 계좌: $19.10 (1번 - 0.9)
...
20번 계좌: $14.30 (1번 - 5.7)
```

#### **매수 간격 설정**
- **간격**: $0.3 (경험적 최적값)
- **하락 대응**: 단계적 매수로 평균단가 하향 조정
- **상승 대응**: 일부 계좌만 매수되어 리스크 제한

---

### 📉 **매도 전략**

#### **목표 수익률**
- **기본 수익률**: 5%
- **매도 방식**: LOC(지정가) 매도
- **매도 기간**: 14일 예약매도 (편의성 고려)

#### **매도 가격 계산**
```
1번 계좌: $20.00 × 1.05 = $21.00
2번 계좌: $19.20 × 1.05 = $20.16
3번 계좌: $19.20 × 1.05 = $20.16
```

---

### 🔄 **일일 매매 루틴**

#### **매도 주문**
1. 전날 매수된 계좌들의 목표가로 LOC 매도
2. 14일 예약매도 설정

#### **매수 주문**
1. 가장 낮은 매수단가 기준으로 새로운 등차수열 생성
2. 미매수 계좌들에 LOC 매수 주문

#### **실시간 관리**
- **폭락 시**: 계좌 수익률 1% 달성 시 즉시 매도
- **리스크 관리**: 연속 매수 계좌 3개 이상 시 일부 매도

---

### ⚠️ **리스크 관리**

#### **폭락 대응**
```
상황: 2,3,4,5번 계좌 모두 $15에 매수
대응: 2번 제외하고 3,4,5번 계좌 1% 수익 시 즉시 매도
이유: 계좌 분할의 장점(리스크 헷지) 퇴색 방지
```

#### **상승 대응**
- 일부 계좌만 매수되어 리스크 제한
- 매수된 계좌들의 목표가 매도로 수익 실현

---

### 📊 **전략의 장점**

#### **1. 리스크 분산**
- 20개 계좌로 손실 분산
- 개별 계좌 손실이 전체에 미치는 영향 최소화

#### **2. 변동성 활용**
- SOXL의 3배 레버리지로 변동성 수익 극대화
- 등차수열 매수로 하락 시 평균단가 하향 조정

#### **3. 자동화 가능**
- LOC 주문으로 자동 매매 가능
- 예약매도로 관리 부담 최소화

#### **4. 유연한 대응**
- 시장 상황에 따른 실시간 조정 가능
- 폭락 시 즉시 매도로 추가 손실 방지

---

### 🎯 **핵심 성공 요소**

1. **일관성**: 매일 동일한 루틴 실행
2. **규율**: 감정적 매매 금지
3. **모니터링**: 실시간 수익률 체크
4. **적응성**: 시장 변화에 따른 전략 조정

이 전략은 체계적이고 논리적인 접근으로 단타 매매의 리스크를 최소화하면서도 수익 기회를 극대화하는 훌륭한 방법입니다! 🚀
